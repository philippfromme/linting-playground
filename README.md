### Query Parameters

| Parameter | Values |
|---|---|
| `number` | `0` or `1` |
| `opacity` | `0` or `1` |
| `position` | `top` or `bottom` |
| `variant` | `1`, `2`, `3` |

Example: http://philippfrom.me/linting-playground?number=0&opacity=1&position=bottom&variant=1
